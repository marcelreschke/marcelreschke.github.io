# UiKit-Sketching :snake::money_with_wings::bank:

HTML5 Sketching with UiKit

## Installation


    $ git clone https://github.com/marcelreschke/uikit-sketching.git
    $ cd uikit-sketching/
    $ npm install
    $ bower install
    $ gulp
